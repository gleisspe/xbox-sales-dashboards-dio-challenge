📊 Dashboard de Vendas Xbox

Este projeto apresenta um dashboard interativo de vendas focado em produtos Xbox, desenvolvido com o objetivo de transformar dados brutos em insights estratégicos.

O principal objetivo foi analisar o desempenho de vendas, identificar tendências e apoiar a tomada de decisão baseada em dados. O dashboard destaca métricas importantes como faturamento, volume de vendas e performance de produtos, oferecendo uma visualização clara e intuitiva das informações.

Durante o desenvolvimento, apliquei técnicas de análise e visualização de dados, fortalecendo habilidades em limpeza, organização e storytelling com dados. Este projeto reflete meu interesse em conectar visão de negócio com ferramentas analíticas para gerar valor e apoiar decisões estratégicas.

🚀 Principais Funcionalidades:
Análise de desempenho de vendas
Insights de faturamento e produtos
Design limpo e intuitivo
Storytelling orientado por dados

📊 Xbox Sales Dashboard

This project presents an interactive sales dashboard focused on Xbox products, developed to transform raw data into actionable business insights.

The main goal was to analyze sales performance, identify trends, and support data-driven decision-making. The dashboard highlights key metrics such as revenue, sales volume, and product performance, providing a clear and intuitive visualization of the data.

Through this project, I applied data analysis and visualization techniques, reinforcing skills in data cleaning, organization, and storytelling with data. This work reflects my interest in combining business understanding with analytical tools to generate insights and drive strategic decisions.

🚀 Key Features:
Sales performance analysis
Revenue and product insights
Clean and intuitive dashboard design
Data-driven storytelling
